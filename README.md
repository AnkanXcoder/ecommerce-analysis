# 📊 E-commerce Customer & Revenue Analysis

## 📌 Overview

This project analyzes an e-commerce transaction dataset using Python and Pandas to extract meaningful business insights. The analysis includes data cleaning, feature engineering, customer segmentation, and revenue analysis.

---

## 🔧 Key Steps

* Data cleaning and handling missing values
* Feature engineering (TotalPrice, datetime features)
* Customer segmentation and behavior analysis
* Cohort analysis to track customer retention
* Revenue trends and cumulative revenue analysis

---

## 📈 Key Insights

* Identified top customers and high-value segments
* Analyzed country-wise revenue performance
* Tracked customer retention using cohort analysis
* Measured cumulative revenue growth over time

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy

---

## 📂 Dataset

Online Retail Dataset (UCI Machine Learning Repository)

---

## 🎯 Objective

To understand customer behavior, identify revenue drivers, and generate actionable insights from raw transaction data.
